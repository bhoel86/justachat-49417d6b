# VPS ISSUES LOG

Each file in this directory documents a specific issue encountered on the VPS.

## Format
Every issue file follows this template:

```
# ISSUE: <Short Title>
Date: YYYY-MM-DD

## Problem
What went wrong.

## Symptoms
What the user saw (errors, blank pages, etc.)

## Root Cause
Why it happened.

## Solution
Step-by-step fix.

## Prevention
How to avoid it in the future.
```
